ABC
===

|docs|

Massively parallel ABC for Python.


Building the documentation
--------------------------

Make sure to set your `PYTHONPATH` environment
variable accordingly. Then run `make html` in
the `doc` folder. The html doc can be found in
`doc/_build/html`.
